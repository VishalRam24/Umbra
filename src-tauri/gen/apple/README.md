# Apple Mobile Native Scaffold

This folder is intentionally tracked as part of the repository skeleton.

On a macOS machine with Xcode installed, initialize and generate the full iOS native project with:

```bash
npm run tauri ios init
```

Then build with:

```bash
npm run build:ios
```

Expected generated output will be under `src-tauri/gen/apple/` and `src-tauri/target/`.
