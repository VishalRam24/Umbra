# Android Native Scaffold

This folder is intentionally tracked as part of the repository skeleton.

On a machine with Android SDK/NDK and Java installed, initialize and generate the full Android native project with:

```bash
npm run tauri android init
```

Then build with:

```bash
npm run build:android
```

Expected generated output will be under `src-tauri/gen/android/` and `src-tauri/target/`.
